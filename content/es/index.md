# Introducción (ES)

Esta página es un **stub de i18n** para demos empresariales. El contenido canónico en inglés vive en [Introduction](/index).

Hullo es una CLI de un solo binario que sirve APIs HTTP simuladas a partir de un archivo `hullo.yaml`.

## Qué demuestra este stub

- Una carpeta de locale (`content/es/`) con su propio `_section.json`
- Metadatos `locale: es` en el frontmatter
- Enlaces cruzados al idioma principal

Para un sitio multi-idioma de producción, cada locale tendría su árbol completo y un selector de idioma en el lector.
